## 二、核心指标体系

### 2.1 衰退预测指标

#### 指标1：2s10s利差（美国国债2年与10年期利差）

| 属性 | 内容 |
|------|------|
| 定义 | 10年期美债收益率 - 2年期美债收益率 |
| 计算公式 | `spread_2s10s = yield_10Y - yield_2Y` |
| 数据来源 | FRED: T10Y2Y 或 akshare: bond_zh_us_rate() |
| 更新频率 | 每日 |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| > +1.0% | 🟢 正常 | 收益率曲线陡峭，经济扩张 |
| +0.5% ~ +1.0% | 🟡 关注 | 曲线趋平，经济增长放缓 |
| 0% ~ +0.5% | 🟠 预警 | 曲线极度平坦，衰退风险上升 |
| < 0% | 🔴 危险 | 曲线倒挂，衰退信号（历史准确率100%） |

**历史案例：**
- 2000年倒挂 → 2001年互联网泡沫
- 2006年倒挂 → 2008年金融危机
- 2019年倒挂 → 2020年疫情衰退

---

#### 指标2：10s30s利差

| 属性 | 内容 |
|------|------|
| 定义 | 30年期美债收益率 - 10年期美债收益率 |
| 计算公式 | `spread_10s30s = yield_30Y - yield_10Y` |
| 数据来源 | FRED: T10Y30Y 或 akshare |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| > +0.5% | 🟢 正常 | 长端溢价正常 |
| 0% ~ +0.5% | 🟡 关注 | 长端溢价压缩 |
| < 0% | 🟠 异常 | 长端倒挂，极度避险情绪 |

---

### 2.2 流动性危机指标

#### 指标3：TED利差

| 属性 | 内容 |
|------|------|
| 定义 | 3个月LIBOR - 3个月美国国债收益率 |
| 计算公式 | `TED = LIBOR_3M - TBill_3M` |
| 数据来源 | FRED: TEDRATE |
| 经济含义 | 衡量银行间市场信用风险，反映金融体系流动性 |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| < 0.5% | 🟢 正常 | 流动性充足 |
| 0.5% ~ 1.0% | 🟡 关注 | 流动性趋紧 |
| 1.0% ~ 2.0% | 🟠 预警 | 银行间市场紧张 |
| > 2.0% | 🔴 危机 | 流动性危机（如2008年） |

**历史案例：**
- 2008年10月：TED利差飙升至4.5%，全球流动性冻结

---

#### 指标4：VIX恐慌指数

| 属性 | 内容 |
|------|------|
| 定义 | 标普500指数期权隐含波动率（30天） |
| 计算公式 | 基于S&P 500期权价格计算 |
| 数据来源 | akshare: index_us_stock_sina('.VIX') 或 CBOE |
| 经济含义 | 市场对未来30天波动率的预期 |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| < 15 | 🟢 极度贪婪 | 市场平静，适合风险资产 |
| 15 ~ 20 | 🟢 正常 | 波动率正常水平 |
| 20 ~ 30 | 🟡 关注 | 恐惧上升，需谨慎 |
| 30 ~ 40 | 🟠 预警 | 市场恐慌，避险情绪浓厚 |
| > 40 | 🔴 极度恐慌 | 历史性恐慌，往往是买入机会 |

---

#### 指标5：MOVE指数（美债波动率指数）

| 属性 | 内容 |
|------|------|
| 定义 | 美国国债市场隐含波动率指数 |
| 经济含义 | 债市恐慌程度，类似VIX但针对债券 |
| 数据来源 | ICE/Bloomberg（需付费） |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| < 80 | 🟢 正常 | 债市稳定 |
| 80 ~ 100 | 🟡 关注 | 债市波动上升 |
| 100 ~ 150 | 🟠 预警 | 债市不稳定 |
| > 150 | 🔴 危机 | 债市恐慌（如2020年3月） |

---

### 2.3 市场情绪指标

#### 指标6：信用利差（高收益债 - 国债）

| 属性 | 内容 |
|------|------|
| 定义 | 高收益企业债收益率 - 同期限国债收益率 |
| 数据来源 | FRED: BAMLC0A0CM（ICE BofA高收益债利差） |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| < 3% | 🟢 风险偏好 | 资金追逐高收益 |
| 3% ~ 5% | 🟡 正常 | 风险定价合理 |
| 5% ~ 8% | 🟠 预警 | 信用风险上升 |
| > 8% | 🔴 危机 | 信用市场冻结 |

---

### 2.4 全球宏观指标

#### 指标7：美元指数DXY

| 属性 | 内容 |
|------|------|
| 定义 | 美元对一篮子货币的加权汇率指数 |
| 数据来源 | akshare: futures_zh_spot(symbol='美元指数') |

**阈值判断：**
| 区间 | 状态 | 含义 |
|------|------|------|
| < 95 | 🟢 弱美元 | 利好新兴市场、大宗商品 |
| 95 ~ 100 | 🟡 正常 | 中性 |
| 100 ~ 105 | 🟠 偏强 | 新兴市场承压 |
| > 105 | 🔴 强美元 | 全球流动性收紧，新兴市场危机风险 |

## 三、投资风格判断逻辑

### 3.1 风险偏好（Risk-On）触发条件

```
当以下条件满足3个以上时，判定为风险偏好：
✅ VIX < 20
✅ 2s10s利差 > +0.5%
✅ 信用利差 < 4%
✅ TED利差 < 0.5%
✅ DXY < 100
```

**建议操作：**
- 增配股票（尤其是成长股、科技股）
- 减持债券和黄金
- 加杠杆（适度）

---

### 3.2 避险模式（Risk-Off）触发条件

```
当以下条件满足3个以上时，判定为避险模式：
✅ VIX > 30
✅ 2s10s利差 < 0%（倒挂）
✅ 信用利差 > 6%
✅ TED利差 > 1%
✅ DXY > 105
```

**建议操作：**
- 减持股票（尤其是高估值成长股）
- 增配国债、黄金、日元
- 降低杠杆
- 持有现金
